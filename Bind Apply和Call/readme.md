# 1. Apply()/Call()

    每个函数都包含两个非继承而来的方法,apply() 和 call(). apply()方法接收两个参数,一个是在其中运行函数的作用域,
    另一个是参数数组.
    
    apply() 和 call() 真正强大的地方在于能够扩充函数赖以运行的作用域.
    使用 apply() 和 call() 来扩充作用域的最大好处,就是对象不需要与方法有任何耦合关系。
